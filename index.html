List of git commands:

  • git clone - clone repository
  • git commit -m <message> – make commit containing a message
  • git status - check the status of the files in the branch
  • git add - add files to a staging area
  • git checkout -b <branch name> - create and switch to a new branch
  • git branch - display the name of the current and other branches
  • git log - display commit history
  • git push origin HEAD - push code to a remote repository with a HEAD commit
